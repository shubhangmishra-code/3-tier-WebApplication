## AWS Deployment Notes

For deployment, ensure:
- Backend environment variables in `app-tier/DbConfig.js` are configured for your AWS database (for example, RDS).
- The backend is reachable by the web tier (routing/proxy and security group rules).
- The web tier serves the production build generated by `npm run build`.

Because the backend has no automated tests yet, validate backend health after deployment using:
- `GET /health`
- Transaction CRUD endpoints under `/transaction`
